03 À propos

Cette démo effectue toutes les opérations directement dans votre navigateur contre votre instance VexObj. La clé API reste en local (localStorage) et n'est jamais envoyée à un tiers.

Sources : site/demo/index.html.

Contenu mixte bloqué. Cette page est servie en HTTPS, mais votre instance est en HTTP. Pour tester en local, clonez le dépôt et servez site/demo/ localement, ou déployez votre instance VexObj derrière TLS.
Bienvenue. Démarrez VexObj localement avec docker run -p 8000:8000 ghcr.io/aniasse/vexobj:latest, récupérez la clé admin dans les logs, et saisissez vos informations de connexion dans le panneau de gauche.
